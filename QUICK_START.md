# 🚀 Globa7 Quick Start Checklist

Use this checklist to get your platform up and running quickly.

---

## ✅ Step 1: Initial Setup (5 minutes)

- [ ] Platform is loaded and running
- [ ] Can access home page at `/`
- [ ] Can see all sections (Hero, Services, Fleet, etc.)
- [ ] Navigation works (scrolling to sections)
- [ ] "Reserve Now" button opens modal

---

## ✅ Step 2: Create Your First Admin Account (2 minutes)

- [ ] Navigate to `/auth`
- [ ] Click "Sign Up" tab
- [ ] Enter admin details:
  - Name: Your Name
  - Phone: Your Phone
  - Email: your-admin@email.com
  - Password: Strong Password
- [ ] Click "Create Account"
- [ ] Successfully redirected to portal

**⚠️ Important**: This account needs to be promoted to admin role manually.

---

## ✅ Step 3: Test Client Features (5 minutes)

- [ ] View client dashboard at `/portal`
- [ ] See stats cards (Bookings, Upcoming, Completed, Points)
- [ ] Navigate between tabs (Bookings, Profile, Addresses, Rewards)
- [ ] Click "Edit" on profile
- [ ] Update profile information
- [ ] Click "Save Changes"
- [ ] Verify changes persist

---

## ✅ Step 4: Create a Test Booking (3 minutes)

- [ ] Go back to home page `/`
- [ ] Click "Reserve Now"
- [ ] Fill out booking form:
  - Service Type: Airport Transfer
  - Pickup: Test Location
  - Dropoff: Test Destination
  - Date/Time: Tomorrow
  - Passengers: 2
  - Vehicle: Luxury Sedan
  - Contact info
- [ ] Submit booking
- [ ] See success message
- [ ] Check booking in portal

---

## ✅ Step 5: Access Admin Dashboard (2 minutes)

⚠️ **Note**: You need admin role to access this section.

- [ ] Navigate to `/admin`
- [ ] If redirected to portal, role is not admin yet
- [ ] If successful, see admin dashboard
- [ ] View analytics cards
- [ ] See the test booking you created

---

## ✅ Step 6: Configure Pricing (5 minutes)

- [ ] In admin dashboard, click "Pricing" tab
- [ ] Click "Edit Pricing"
- [ ] Update base rates:
  - Sedan: $75 (or your rate)
  - SUV: $95 (or your rate)
  - Luxury: $150 (or your rate)
  - Sprinter: $200 (or your rate)
- [ ] Review zone multipliers
- [ ] Review time multipliers
- [ ] Click "Save"
- [ ] Verify success message

---

## ✅ Step 7: Add Your First Driver (3 minutes)

- [ ] In admin dashboard, click "Drivers" tab
- [ ] Click "Add Driver"
- [ ] Enter driver details:
  - Name: Driver Full Name
  - Phone: (504) 555-0123
  - Email: driver@globa7.com
  - Vehicle: 2023 Mercedes S-Class
- [ ] Click "Add Driver"
- [ ] Driver appears in list
- [ ] Test status change (Available/Busy/Offline)

---

## ✅ Step 8: Create Your First Bundle (3 minutes)

- [ ] In admin dashboard, click "Bundles" tab
- [ ] Click "Create Bundle"
- [ ] Enter bundle details:
  - Name: Wedding Package
  - Description: Premium wedding day transportation
  - Price: 1500
  - Discount: 15
- [ ] Click "Create Bundle"
- [ ] Bundle appears in list

---

## ✅ Step 9: Test Booking Management (3 minutes)

- [ ] In admin dashboard, click "Bookings" tab
- [ ] Find your test booking
- [ ] Change status from "pending" to "confirmed"
- [ ] Assign the driver you created
- [ ] Verify changes save

---

## ✅ Step 10: Mobile Testing (5 minutes)

- [ ] Resize browser to mobile width (< 768px)
- [ ] Test mobile navigation menu
- [ ] Check bottom action bar appears
- [ ] Test "Quick Call" and "Book Now" buttons
- [ ] Verify all pages are mobile-friendly:
  - [ ] Home page
  - [ ] Auth page
  - [ ] Client portal
  - [ ] Admin dashboard

---

## ✅ Step 11: Test Sign Out/In Flow (2 minutes)

- [ ] Click sign out button
- [ ] Redirected to home page
- [ ] Click "Sign In"
- [ ] Enter credentials
- [ ] Successfully sign back in
- [ ] Correct portal opens (client or admin)

---

## ✅ Step 12: Review Analytics (2 minutes)

- [ ] In admin dashboard, view analytics cards
- [ ] Verify "Total Bookings" shows 1 (your test booking)
- [ ] Check "Pending Bookings" count
- [ ] Review "Total Revenue"
- [ ] Verify numbers match expectations

---

## ✅ Step 13: Test Multiple Users (5 minutes)

- [ ] Open incognito/private window
- [ ] Create second client account
- [ ] Make another test booking
- [ ] Check if it appears in admin dashboard
- [ ] Test that users only see their own bookings in portal

---

## ✅ Step 14: Verify Persistence (2 minutes)

- [ ] Refresh the page
- [ ] Check if still logged in
- [ ] Verify bookings still there
- [ ] Check pricing still saved
- [ ] Verify drivers still listed
- [ ] Confirm bundles persist

---

## ✅ Step 15: Final Visual Check (3 minutes)

- [ ] All gold accents (#D4AF37) displaying correctly
- [ ] Dark theme consistent throughout
- [ ] Animations smooth and professional
- [ ] Typography looks premium (Cormorant + Montserrat)
- [ ] No broken layouts
- [ ] Images/icons all loading
- [ ] Buttons have hover effects
- [ ] Forms are properly styled
